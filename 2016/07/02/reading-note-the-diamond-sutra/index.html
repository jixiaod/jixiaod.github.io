［读书笔记］－ 《金刚经》

发表于 2016-07-02   |   分类于 读书笔记

《金刚经》全文其实只有一个字，“空”。既不能执着诸法的实有，也不能执着一切皆空。

每个人的人生，都充满了起伏不定，喜怒哀乐，这一切看似实实在在，但是它并不是人生的真相。只有断除对万法的执着，通达本来无我，才会懂得人生到底是什么。

我们痛苦的根源，就是执着。哪怕只有一点点执念，一开始只有一点点，也好似雪球一样会越滚越大，且会绵绵不断。对于执着的东西，得到了就会患得患失。失去了，就会伤心难过。假如没有执着，任何痛苦就会与你无缘。【不管是福德也好、法身也好，我们对任何一个法都不能执着，如此才可了悟真实的空性】。

不过执着单独的空性，也会落于四相当中。因为认为空性存在，显然是一种我所执（人相），有了我所执就一定有我执（我相），进而人我的分别也会产生（众生相），既然人我的相都有了，那有情从生至死的现象不可能不存在（寿者相）。

因此，与其让愚者有“我”的执着，也不可让他有“无我”的空性道理。因为若是执着“我”，还可以用空性对治，但若执着空性，这是谁也救不了的。

如筏喻者，渡河时离不开船筏，但到了对岸后，船筏就没有用处了，就必须舍去。【是故，不应取法，不应取非法】。因此，既不能执着诸法的实有，也不能执着一切皆空。

【故，如来常说：汝等比丘，知我说法，如筏喻者，法尚应舍，何况非法？】

当然还有，“若不依俗谛，不得第一义”。就像还没有到岸便不能轻易抛弃船筏，否则，半途掉到大海里，后果是很可怕的！

【一切有为法，如梦幻泡影，如露亦如电，应作如是观】。不应该执着任何相，了知一切皆如梦如幻。【唐朝法融禅师说：恰恰用心时，恰恰无心用】。正在专注的时候，没有什么可专注的，同样在执着的时候，也没什么可执着的，应该有这样的智慧。

扫一下呗，反正我也不怎么发～
