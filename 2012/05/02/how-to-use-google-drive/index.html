google drive 使用

发表于 2012-05-02   |   分类于 杂

google刚推出了自己的云存储，试试。搜到了一个drive的IP地址

绑定google doc && drive IP host， 都用一个地址。

刚登录发现其中默认存储了我之前的google note(被google阉割的项目)的笔记，之前在docs中也有保留记录。

测试上传78.3M的文件，上传速度大概200～300kb/s。用的公司的网络，还不错。大概5分钟搞定了。这速度比dropbox好点吧，可能是目前使用用户还不多的缘故。

界面很简洁，一直比较热衷google 的界面风格。

扫一下呗，反正我也不怎么发～
