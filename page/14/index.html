沈阳耐斯一周

发表于 2010-02-05   |   分类于 杂

还有几天就回家了，这几天在这鼓捣了一个多星期的j2me。感觉作游戏是一个比较逻辑行很强的工作，而且需要很大的耐心。话说回来，什么工作不需要这些呢。现在虽说作上手机上面的开发了，但是，这几天想了想，觉得游戏在手机上没有什么可玩性。比较起在PC上，手机上的游戏操作性和可玩性都受到一定的局限。具体现在来分析，还是不太能分析权衡利弊，到底怎么样才是好的。

今天在校内分享看了一个TED的一期节目，美国MIT的一位印度小伙。他的发明，可以说是一种将来的一种模式。计算机随身带－－虚拟和现实的结合，计算机与人的结合。在手指上带上不同颜色的套子，摆一个照相的矩形手势，就会自动识别照相，拍下眼前的风景。这更可以说是体验式的。

体验式的可能会成为将来的流行，原来公司老板(虽然不懂技术)也说要做体验式的电子商务。这种体验式的，现实与虚拟结合的东西一定会成为一中趋势。比起现在，堆积在垃圾代码之上的表面虽然很美观的网页。我们或许更需要一种语言，一种更便捷的语言。html5已经出现，他已经包含了flash的支持。以后可能就不再需要，安查检就可以播放。或者直接在html中嵌入3D。

道不是向Jobs说的那样，Adobe的人懒。因为谁都想作自己的规范，我不会为了别人而改变。就像MS不会开放源代码，或者作linux的东西一样。

还有就是作软件，不只是要学会技术。还要学会分享，学会传授，表达自己的想法、思想。当年的linux操作系统创始人linux，如果没有很好的表达自己的想法。只是自己作出原来linux的雏形，linux也不会有今天的发展。所以，一定要学会传达、传递自己的思想。说出有感染力的话，感然别人。

Android 安装

发表于 2010-02-03   |   分类于 杂

1)解压Android SDK
执行如上目录下的文件：SDK Setup.exe，（机器人图标的哪个）
注意：一般的话会出现 Failed to fetch URL https://dl-ssl.google.com/android/repository/repository.xml，所以应该修改如下几个地方：
a)修改Available Packages,点击”add Site..”,增加: http://dl-ssl.google.com/android/repository/repository.xml,当然了，你可以删除原来使用https做连接的site.这个只能更新一些API的package.
b)修改代理配置：Settings里Http Proxy Server:10.159.192.62,Http Proxy Port:8080,选择如下两个选项卡：Force https//…..using http…,还有一个：Ask Before restaring ADB.
c)选择Installed package,里面会有SDK1.1,1.5,1.6,2.0和多个版本的APIs及Usb Driver package.
至此，所有SDK安装完毕。CALL，也不知中国为什么封技术性的网站，按照如上做能够省出你更多的时间来。

2)安装Google为我们提供的Eclipse 开发插件(ADT),地址：https://dl-ssl.google.com/android/eclipse/

ubuntu 配置 j2me 环境 wtk2.5.2

发表于 2010-01-31   |   分类于 linux

PS:已经安装jdk 1.6，eclipse

1.下载eclipseme.zip && wtk2.5.2 (java sun 的linux版)

2.eclipse -> help ->install new software **（略)，添加eclipseme.zip。

3.wtk2.5.2 安装：

环境需求，其实不知道用不用，不过我都sudo apt-get install ** 安装了。
System Requirements - Software
Microsoft Windows XP and Linux-x86 (English version tested with Ubuntu 6.x)
The following libraries should be present on the system running Ubuntu v6.x:
ibXpm (libxpm-dev)
libXt (libxt-dev)
libX11 (libx11-dev)
libICE (libice-dev)
libSM (libsm-dev)
libpthread (libc6-dev)
libm (libc6-dev)
libnsl (libc6-dev)
libstdc++6-dev
JavaTM 2 Platform, Standard Edition (Java SE SDK), version 1.5.0 - if you plan to do development, or JavaTM 2, Standard Edition Runtime Environment (JRE), version 1.5.0 - if you only plan to run the demonstration applications.
To download the SDK or JRE you want, go to http://java.sun.com/javase/downloads/index.html
Apple QuickTime player (required to play AMR media on Windows)
To play Adaptive Multi-Rate (AMR) content on Linux you must have a 3GPP reference implementation. Chapter 8 of the Sun Java Wireless Toolkit User’s Guide describes how to enable AMR support.

下载的wtk2.5.2 为
sun_java_wireless_toolkit-2.5.2_01-linuxi486.bin.sh 文件

~#sudo chmod +x sun_java_wireless_toolkit-2.5.2_01-linuxi486.bin.sh
~#sudo ./sun_java_wireless_toolkit-2.5.2_01-linuxi486.bin.sh

中间过程会让你选择，jdk/bin所在路径 ，还有设置安装路径 。安装路径设置在/home/*目录下，否则安装到最后会提示用户权限不够。

4.eclipse 添加 wtk：
eclipse -> windows -> Preferences -> J2ME ->Divce Managerment ,import 你刚才 安装wtk的目录。refresh - 会出现几个 divce,不用考虑，都添加就好了。

5.新建一个j2me项目看看。。

看不出的骗子

发表于 2010-01-22   |   分类于 杂

今天看见一句话，“流氓不分年龄”。让我想起在广州火车站的事儿，为了赶到沈阳北的火车，跟一个票贩子被警察逮到了派出所（当然不管我的事，留了身份证号和电话就让我走了）。就在所里面聊天的空，一个长相中肯、看上去很老实的人给抓进来了。警察好像对他很熟悉和了解了，“怎么招？掏出来吧！还等什么呢？”我明白了，原来是小偷，今天我遇上真的了。“行啊，又换钱包了，刚才还不是这个呢。”警察边看钱包，边对他说，显然已经见怪不怪了。我偷笑……那人还狡辩“没换，一直都这个！真没换……”无奈猫技高一筹，从钱包里翻出了身份证“你看这是你吗？行了，行了……”那人哑火，不说话了。

再仔细打量一下这个人，真的看不出来是小偷。50岁左右，皮肤很好，只有几条不算明显的皱纹能告诉人他已经不年轻；黑色夹克，背有些弓：两只手一直揣在夹克口袋里，低着头，很老实的样子。要不是在派出所，打死我都认不出来。不知道是他伪装的好，还是当小偷的就天生一副能骗人的老实像。

出来混的，难免受骗。不光别人较劲心思骗你上钩，自己有时候都傻糊糊心甘情愿上当。要不说人善被人欺，前几天一个收旧家电的都欺负我。好好的热水器，就给20块钱。都怪我太老实，人家验货的时候，说漏了嘴“热水器有点漏水”（其实就是外面的管子漏，找人修下就好）。刚说完这话，大姨就冲我使眼色，我知道又说错话了。事后有被大姨数落了一番，“作人不能太老实了，老实人受欺负！”我深深的认识到我错了，错的无可救药。我决心要改！

真希望有双孙悟空的火眼金睛，看清妖魔、分辨善恶。妖魔者，除之杀之；真善者，亲之近之。有歌词写得好“借我借我一双慧眼吧”，雾里看花，的确需要一双好眼睛。好眼睛光滴眼药水是没用的，需要修炼。想当年孙悟空不也是在太上老君的炼丹炉里受尽磨难才获得，虽然是小说胡编乱造。但是不无道理，一双金睛又岂能是谁都能有、谁都配有的。必须在社会上摸爬滚打，潜心修行。

我本将心向明月,奈何明月照沟渠。我想我除了需要一双辨别善恶的眼睛，还需要一张能言善辩的伶牙俐齿。
